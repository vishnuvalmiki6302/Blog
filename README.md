# Project Title: Full-Stack Personal Portfolio

A modern, responsive, and performance-optimized personal portfolio built to showcase professional achievements, skills, and projects.

## 🚀 Live Demo
- **Frontend**:https://vishnu-vardhan-five.vercel.app
- **Backend**: https://blog-sigma-six.vercel.app

## 🛠 Features
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
- **Dynamic Projects Section**: Alternating layout for project items on desktop.
- **Smooth Navigation**: Custom smooth scrolling with Lenis for an enhanced UX.
- **Contact Form**: Integrated backend with Nodemailer for direct email responses.
- **Production Ready**: Configured for seamless deployment on Vercel as a monorepo.

## 💻 Tech Stack
### Frontend
- **React.js**: Modern component-based architecture.
- **Vite**: Ultra-fast build tool and development server.
- **Tailwind CSS**: Utility-first styling with modern aesthetics.
- **Studio Freight Lenis**: For smooth scrolling experience.

### Backend
- **Node.js & Express**: Scalable backend infrastructure.
- **Nodemailer**: For handling contact form email processing.
- **Vercel Serverless Functions**: Effortless backend deployment.

## 📦 Setup & Installation
### Prerequisites
- Node.js (v18+)
- npm

### 1. Clone the repository
```bash
git clone https://github.com/vishnuvalmiki6302/Blog.git
cd Blog
```

### 2. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### 3. Backend Setup
```bash
cd ../backend
npm install
node server.js
```

### 4. Environment Variables
Create a `.env` file in the `backend` folder:
```bash
EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-gmail-app-password
```
